# MatchaLog

A discovery and review platform for matcha enthusiasts to explore products, track their collection, and share experiences.

🔗 [Live Demo](https://matchalog-4ryqxmywm-jennyca0s-projects.vercel.app/) 

## Features

- 🔍 **Product Discovery** - Browse and search matcha products with real-time filtering
- ⭐ **Community Reviews** - Read and write reviews with ratings
- 📦 **Personal Stash** - Track your matcha collection  
- 🔐 **User Authentication** - Secure auth with Supabase

## Tech Stack

**Frontend:** React, Next.js, CSS Modules  
**Backend:** Next.js API Routes, Supabase  
**Database:** PostgreSQL (Supabase)  
**Auth:** Supabase Auth  
**Deployment:** Vercel

## What I've Learned So Far

- Implementing secure authentication flows with JWTs and session management
- Designing relational database schemas for social features
- Building real-time search with debouncing for performance
- Managing state across complex user interactions
